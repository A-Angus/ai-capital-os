## Deal Stage Definitions

| Stage | Meaning |
|-------|---------|
| 🔵 Intake / Screening | Deal received, running through deal score criteria |
| 🟡 Structuring | Deal passes screening, capital stack being built |
| 🟠 Submitted to Lender | Package sent, awaiting lender response |
| 🟢 Under Review | Lender or capital source actively reviewing |
| ✅ Funded | Deal closed, fee earned |
| ❌ Dead | Deal fell through or declined |

## Deal Intake Checklist

Use this checklist when a new deal comes in.

- [ ] Borrower name
- [ ] Borrower credit score
- [ ] Borrower liquidity
- [ ] Property address
- [ ] Property type
- [ ] Purchase price
- [ ] Rehab budget
- [ ] Estimated ARV
- [ ] Loan amount requested
- [ ] Exit strategy (flip / refinance / rental hold / development)
- [ ] Timeline to close
